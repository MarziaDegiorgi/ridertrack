What is Ridertrack?

Rider Track is an user-friendly management service for outdoor sport events, which allows spectators, competitors and organizers to live an easy and engaging race experience. The main peculiarity of the system is to support multiple input sources from different tracking devices.

What services are provided to a not registered user?

    A user not already registered can:

  1. Surf the home page
  2. Consult the list of all the events
  3. Attend a race as a spectator and track whichever event through the web site
  4. Get in contact with the favourite entrant, by means of the secret code

What additional services are provided to a registered user?

    A registered user can:

  1. Log in and access his/her personal area
  2. Manage personal data
  3. See the list of the events in which he/she is already enrolled
  4. See the list of the events to come, which he/she has organized
  5. See the list of the past events in which he/she took part
  6. See the list of the past events which he/she organized
  7. Enroll to a future event
  8. Create and organize a new event

Can anyone organize an event?

That's for sure! All you need is a nice location, the perfect date and the will to enjoy yourself!

Can I unenroll from an event?

Yes, you can unenroll from an event in whichever moment, before the enrollment closing time.

What is the Ridertrack mobile application?

The Ridertrack mobile application is an additional service provided to athletes competing in a race managed by Ridertrack, which enable the participant to:

  1. Provide more detailed data about his/her progresses and rhythm during the race
  2. Receive messages from authorized spectators and listen to them thought his/her earphones while racing
  3. Take constantly under control the time passed, the average speed and the distance to the finish line
  4. Instantly share the listed above details about the race on Facebook, with just a click
